# Healthcare Analytics Dashboard - Power BI

## Project Overview
This project analyzes healthcare and financial data using Power BI dashboards.

The dashboard provides insights into:
- Revenue trends
- Appointment analysis
- Doctor performance
- Treatment analysis
- Payment status analysis

---

## Tools Used
- Power BI
- DAX
- Power Query
- Data Modeling

---

## Dashboard Features
- KPI Cards
- Interactive Slicers
- Conditional Formatting
- Revenue Trend Analysis
- Financial Insights Dashboard
- Doctor Performance Analysis
